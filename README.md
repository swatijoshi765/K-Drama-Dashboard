
K-Drama Insights Dashboard

An interactive and visually appealing dashboard built using **Python** and **Streamlit** to explore and analyze Korean Dramas.  
This project allows users to view **Top Rated Dramas**, explore **genres**, check **platform availability**, and see **cast & story details** at a glance.



Features
- Top 10 Highest Rated K-Dramas (Static List)
- Filter by Genre & Streaming Platform
- Poster Display for Each Drama
- Custom UI Styling using CSS
- Fast & Interactive Streamlit App



Tech Stack
Programming Language - Python
Framework - Streamlit 
Data Handling - Pandas 
Visualization - Matplotlib
Styling - Custom CSS
